#Разработка программного модуля для конфигурирования многопоточного обмена между корпоративными информационными системами Воробьёва А.А.

Итоговый файл для ВКР расположен в разделе **Releases**.

1. Перейдите в **Releases** (находится в правой боковой панели).
2. Найдите тег с названием **`multithreaded_exchange`** .
3. Скачайте прикрепленный файл.
